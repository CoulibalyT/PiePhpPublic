# PIEPHP
PiePHP est un framework home made basé sur la structure de CakePHP et la norme de Symfony.
Ce projet est le projet fondateur du module MVC.

Temps du projet : 3 semaines

# Notion 
PHP – POO
• .htaccess
• Framework MVC
• ORM
• Requêtes HTTP
